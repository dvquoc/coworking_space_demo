## 7. 팀의 강점

| 장점 | 세부 내용 |
|------|----------|
| 🎯 **프로토타입 준비 완료** | 프론트엔드 95% 완성 → 시니어 FE 개발자가 처음부터 만드는 대신 API 연동에 집중 |
| 🔧 **풀스택 역량** | 미드 백엔드가 필요 시 프론트엔드도 유연하게 지원 |
| 📱 **전문 모바일 개발** | 모바일 개발자가 React Native/Flutter 전담, 웹과 시간 분배 없음 |
| 🧪 **조기 QA 참여** | QA가 Sprint 1부터 참여, 개발과 병행하여 테스트 계획 수립 |
| 📋 **정기적 데모** | PM이 2주마다 데모 주관, Cobi가 항상 진행 상황 파악 |

## 8. 품질 보증

- **코드 리뷰**: 모든 코드는 시니어 리뷰 후 머지
- **자동 테스트**: API 유닛 테스트 + 통합 테스트 작성
- **QA 테스트**: 매 Sprint 수동 테스트 + 리그레션 테스트
- **UAT**: 각 오픈 전 Cobi 인수 테스트
- **보증**: 각 Phase 오픈 후 6개월 무상 버그 수정
