## 1. 시스템 개요

시스템은 **45개 이상 화면**으로 구성되며, **10개 기능 그룹**, **6개 사용자 역할**을 지원합니다:

| 역할 | 설명 | 주요 접근 |
|------|------|----------|
| **Admin** | 전체 시스템 관리 | 모든 모듈 |
| **Manager** | 일일 운영 관리 | 예약, 계약, 고객, CRM |
| **Sale** | 영업, 판매 | 예약, CRM, 고객, 견적 |
| **Accountant** | 회계, 재무 | 청구서, 결제, 크레딧, 리포트 |
| **Maintenance** | 유지보수, 기술 | 유지보수, 자산 |
| **Investor** | 투자자 | 종합 대시보드, 리포트 |

## 3. 주요 업무 흐름

### 3.1 예약 & 결제 흐름 (End-to-End)

```
고객 연락
        │
        ▼
┌─────────────────┐
│ 고객 생성       │ ← Sale/Manager가 정보 입력
│                 │   (개인/법인/소속 직원)
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 예약 생성       │ ← 고객, 공간, 시간 선택
│                 │   자동 중복 감지
└────────┬────────┘
         │
         ▼
┌─────────────────┐     ┌──────────────────────┐
│ 청구서 생성     │────▶│ 결제                 │
│ (보증금/전액)   │     │ VNPay/MoMo/ZaloPay   │
└────────┬────────┘     │ 계좌이체/현금/크레딧  │
         │              └──────────────────────┘
         ▼
┌─────────────────┐
│ 예약 확정       │ ← 결제 후 자동
│                 │   또는 Manager 수동 승인
└─────────────────┘
```

### 3.2 장기 계약 흐름

```
장기 임대 고객
        │
        ▼
┌─────────────────┐
│ 계약 생성       │ ← 템플릿 선택, 정보 입력
│                 │   계약 내용 미리보기
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 고객 확인 전송  │ ← 고객이 약관 확인 & 동의
│                 │   전자 계약
└────────┬────────┘
         │
         ▼
┌─────────────────┐     ┌──────────────────────┐
│ 계약 활성화     │────▶│ 정기 청구서          │
│                 │     │ 월별 자동 생성        │
└────────┬────────┘     └──────────────────────┘
         │
         ▼
┌─────────────────┐
│ 갱신/해지       │ ← 만료 30일 전 알림
│                 │
└─────────────────┘
```

### 3.3 CRM 흐름 (리드 → 고객)

```
다양한 소스의 리드
  (웹사이트, 소개, 이벤트 등)
        │
        ▼
┌─────────────────┐
│ 신규 리드       │ ← 리드 정보 입력
│                 │   담당 Sale 배정
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ Kanban Pipeline │ ← 단계별 드래그 앤 드롭
│ Contacted       │   New → Contacted → Qualified
│ → Qualified     │   → Proposal → Negotiation
│ → Proposal      │   → Won / Lost
│ → Won ✅        │
└────────┬────────┘
         │ (Won)
         ▼
┌─────────────────┐
│ 고객으로 전환   │ ← 자동 고객 프로필 생성
│                 │   → 예약 또는 계약 생성
└─────────────────┘
```
