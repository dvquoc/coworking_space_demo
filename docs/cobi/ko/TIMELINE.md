## 6. Sprint 워크플로우

```
  ┌──────────────────────────────────────────────────────────────┐
  │                      Sprint 2주                              │
  │                                                              │
  │  1주차                              2주차                     │
  │  ┌──────────────────────┐  ┌──────────────────────────────┐  │
  │  │ Planning → Dev → Test│  │ Dev → Test → Review → Demo   │  │
  │  └──────────────────────┘  └──────────────────────────────┘  │
  │                                                              │
  │  📋 Sprint Planning (월요일)        🎯 Cobi 데모 (금요일)    │
  │  💻 개발 + 테스트                   📊 주간 리포트 (금요일)   │
  │  🧪 QA 지속 테스트                  📝 Sprint 회고            │
  └──────────────────────────────────────────────────────────────┘
```

- **2주마다 데모**: Cobi가 직접 진행 상황을 확인하고 즉시 피드백
- **주간 리포트**: 이메일/Slack을 통한 진행 현황, 이슈, 계획 공유
- **Change Request**: 영향 평가 (범위/일정/비용) 후 실행
- **보증**: 각 Phase 오픈 후 6개월 무상 버그 수정
