## 2. Phase 1 비용: 핵심 운영 (5개월)

> 🖥️ Web App — ⭐ 필수

### 포함 범위

| 항목 | 세부 내용 |
|------|----------|
| **백엔드 개발** | 11개 모듈 API, 69+ 기능, 40+ 데이터베이스 엔티티 |
| **프론트엔드 완성** | 프로토타입 45+ 화면에 실제 API 연동 |

| **6개 대시보드** | Admin, Manager, Accountant, Sale, Maintenance, Investor |
| **인프라** | CI/CD, Docker, 스테이징 + 프로덕션 서버 |
| **테스트** | 유닛 테스트, 통합 테스트, UAT |
| **교육** | Cobi 직원 대상 2회 교육 |
