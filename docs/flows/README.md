# Coworking Space – Luồng hoạt động (Flows)

Thư mục này chứa tài liệu mô tả chi tiết từng luồng nghiệp vụ (user flow / business flow) của ứng dụng.

Mỗi flow ghi lại:
- Mục tiêu nghiệp vụ
- Các bước thực hiện (step-by-step)
- Sơ đồ luồng (ASCII)
- Các file code liên quan
- Trạng thái dữ liệu sau mỗi bước

## Quy ước đặt tên file

```
[số thứ tự]-[tên-flow].md

Ví dụ:
  01-auth-flow.md
```

Số thứ tự giúp dễ sắp xếp theo thứ tự ưu tiên / thứ tự phát triển.
