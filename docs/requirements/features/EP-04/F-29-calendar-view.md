# F-29: Calendar View

## Mục tiêu
Xem lịch availability của các không gian.

## Mô tả
- Xem lịch theo ngày/tuần/tháng.
- Màu sắc phân biệt trạng thái: available/booked/maintenance.
- Click vào slot trống để tạo booking nhanh.
- Hover để xem chi tiết booking.

## Acceptance Criteria
- [ ] Calendar view theo ngày/tuần/tháng
- [ ] Filter by building, floor, space type
- [ ] Color-coded: Green (available), Red (booked), Yellow (maintenance)
- [ ] Click vào time slot → quick create booking
- [ ] Hover space → tooltip hiển thị booking details
