## User Stories

### US-76: Generate revenue report
> Là **Nhà đầu tư**, tôi muốn **xem revenue report chi tiết** để **phân tích lợi nhuận**

**Acceptance Criteria**:
- [ ] Select date range, building
- [ ] Report shows: total revenue, by payment method, by space type
- [ ] Chart: Revenue trend
- [ ] Export PDF

### US-77: View occupancy trends
> Là **Manager**, tôi muốn **xem occupancy trend** để **tối ưu pricing**

**Acceptance Criteria**:
- [ ] Chart: Occupancy rate 12 tháng
- [ ] By space type comparison
- [ ] Peak vs off-peak analysis
